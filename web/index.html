Welcome to your Greenstone3 installation!

  • Run the main servlet for Greenstone 3. This is an interface that demonstrates the wide variety of capabilities in Greenstone 3.
  • Run the Greenstone 2 look and feel servlet. (Deprecated) View the demo collections that come with Greenstone, using the Greenstone 2 look and feel.
  • Run the basic servlet. (Deprecated) View the same collections using a basic look and feel.
  • Run the gateway servlet. This uses the standard Greenstone 3 look and feel, and talks via SOAP to the site with the demo collections (localsite).
    • Note that SOAP needs to be installed, and a SOAP server needs to be running for localsite.
  • Run the 'Hello World' test servlet. Use this link to check that your server can run the simplest of Java servlets.
